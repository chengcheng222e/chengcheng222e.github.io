StringBuilderдёәд»Җд№ҲзәҝзЁӢдёҚе®үе…Ё

еҸ‘иЎЁдәҺ 2020-03-25 жӣҙж–°дәҺ 2025-06-25 еҲҶзұ»дәҺ Java

еңЁиҝҷд№ӢеүҚжҲ‘еҸӘи®°дҪҸдәҶStringBuilderдёҚжҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢStringBufferжҳҜзәҝзЁӢе®үе…Ёзҡ„иҝҷдёӘз»“и®әпјҢиҮідәҺStringBuilderдёәд»Җд№ҲдёҚе®үе…Ёд»ҺжқҘжІЎжңүеҺ»жғіиҝҮгҖӮ

еҲҶжһҗ

еңЁеҲҶжһҗи®ҫдёӘй—®йўҳд№ӢеүҚжҲ‘д»¬иҰҒзҹҘйҒ“StringBuilderе’ҢStringBufferзҡ„еҶ…йғЁе®һзҺ°и·ҹStringзұ»дёҖж ·пјҢйғҪжҳҜйҖҡиҝҮдёҖдёӘcharж•°з»„еӯҳеӮЁеӯ—з¬ҰдёІзҡ„пјҢдёҚеҗҢзҡ„жҳҜStringзұ»йҮҢйқўзҡ„charж•°з»„жҳҜfinalдҝ®йҘ°зҡ„пјҢжҳҜдёҚеҸҜеҸҳзҡ„пјҢиҖҢStringBuilderе’ҢStringBufferзҡ„charж•°з»„жҳҜеҸҜеҸҳзҡ„гҖӮ

йҰ–е…ҲйҖҡиҝҮдёҖж®өд»Јз ҒеҺ»зңӢдёҖдёӢеӨҡзәҝзЁӢж“ҚдҪңStringBuilderеҜ№иұЎдјҡеҮәзҺ°д»Җд№Ҳй—®йўҳгҖӮ

жҲ‘д»¬иғҪзңӢеҲ°иҝҷж®өд»Јз ҒеҲӣе»әдәҶ10дёӘзәҝзЁӢпјҢжҜҸдёӘзәҝзЁӢеҫӘзҺҜ1000ж¬ЎеҫҖStringBuilderеҜ№иұЎйҮҢйқўappendеӯ—з¬ҰгҖӮжӯЈеёёжғ…еҶөдёӢд»Јз Ғеә”иҜҘиҫ“еҮә10000пјҢдҪҶжҳҜе®һйҷ…иҝҗиЎҢдјҡиҫ“еҮәд»Җд№Ҳе‘ўпјҹ

жҲ‘д»¬зңӢеҲ°иҫ“еҮәдәҶвҖң9326вҖқпјҢе°ҸдәҺйў„жңҹзҡ„10000пјҢе№¶дё”иҝҳжҠӣеҮәдәҶдёҖдёӘArrayIndexOutOfBoundsExceptionејӮеёёпјҲејӮеёёдёҚжҳҜеҝ…зҺ°пјүгҖӮ

1гҖҒдёәд»Җд№Ҳиҫ“еҮәеҖји·ҹйў„жңҹеҖјдёҚдёҖж ·

жҲ‘д»¬е…ҲзңӢдёҖдёӢStringBuilderзҡ„дёӨдёӘжҲҗе‘ҳеҸҳйҮҸпјҲиҝҷдёӨдёӘжҲҗе‘ҳеҸҳйҮҸе®һйҷ…дёҠжҳҜе®ҡд№үеңЁAbstractStringBuilderйҮҢйқўзҡ„пјҢStringBuilderе’ҢStringBufferйғҪз»§жүҝдәҶAbstractStringBuilderпјү

еҶҚзңӢStringBuilderзҡ„append()ж–№жі•пјҡ

StringBuilderзҡ„append()ж–№жі•и°ғз”Ёзҡ„зҲ¶зұ»AbstractStringBuilderзҡ„append()ж–№жі•

жҲ‘д»¬е…ҲдёҚз®Ўд»Јз Ғзҡ„з¬¬дә”иЎҢе’Ңз¬¬е…ӯиЎҢе№ІдәҶд»Җд№ҲпјҢзӣҙжҺҘзңӢз¬¬дёғиЎҢпјҢcount += lenдёҚжҳҜдёҖдёӘеҺҹеӯҗж“ҚдҪңгҖӮеҒҮи®ҫиҝҷдёӘж—¶еҖҷcountеҖјдёә10пјҢlenеҖјдёә1пјҢдёӨдёӘзәҝзЁӢеҗҢж—¶жү§иЎҢеҲ°дәҶз¬¬дёғиЎҢпјҢжӢҝеҲ°зҡ„countеҖјйғҪжҳҜ10пјҢжү§иЎҢе®ҢеҠ жі•иҝҗз®—еҗҺе°Ҷз»“жһңиөӢеҖјз»ҷcountпјҢжүҖд»ҘдёӨдёӘзәҝзЁӢжү§иЎҢе®ҢеҗҺcountеҖјдёә11пјҢиҖҢдёҚжҳҜ12гҖӮиҝҷе°ұжҳҜдёәд»Җд№ҲжөӢиҜ•д»Јз Ғиҫ“еҮәзҡ„еҖјиҰҒжҜ”10000е°Ҹзҡ„еҺҹеӣ гҖӮ

2гҖҒдёәд»Җд№ҲдјҡжҠӣеҮәArrayIndexOutOfBoundsExceptionејӮеёёгҖӮ

жҲ‘д»¬зңӢеӣһAbstractStringBuilderзҡ„append()ж–№жі•жәҗз Ғзҡ„з¬¬дә”иЎҢпјҢensureCapacityInternal()ж–№жі•жҳҜжЈҖжҹҘStringBuilderеҜ№иұЎзҡ„еҺҹcharж•°з»„зҡ„е®№йҮҸиғҪдёҚиғҪзӣӣдёӢж–°зҡ„еӯ—з¬ҰдёІпјҢеҰӮжһңзӣӣдёҚдёӢе°ұи°ғз”ЁexpandCapacity()ж–№жі•еҜ№charж•°з»„иҝӣиЎҢжү©е®№гҖӮ

жү©е®№зҡ„йҖ»иҫ‘е°ұжҳҜnewдёҖдёӘж–°зҡ„charж•°з»„пјҢж–°зҡ„charж•°з»„зҡ„е®№йҮҸжҳҜеҺҹжқҘcharж•°з»„зҡ„дёӨеҖҚеҶҚеҠ 2пјҢеҶҚйҖҡиҝҮSystem.arryCopy()еҮҪж•°е°ҶеҺҹж•°з»„зҡ„еҶ…е®№еӨҚеҲ¶еҲ°ж–°ж•°з»„пјҢжңҖеҗҺе°ҶжҢҮй’ҲжҢҮеҗ‘ж–°зҡ„charж•°з»„гҖӮ

Arrys.copyOf()ж–№жі•

AbstractStringBuilderзҡ„append()ж–№жі•жәҗз Ғзҡ„з¬¬е…ӯиЎҢпјҢжҳҜе°ҶStringеҜ№иұЎйҮҢйқўcharж•°з»„йҮҢйқўзҡ„еҶ…е®№жӢ·иҙқеҲ°StringBuilderеҜ№иұЎзҡ„charж•°з»„йҮҢйқўпјҢд»Јз ҒеҰӮдёӢпјҡ

getChars()ж–№жі•

жӢ·иҙқжөҒзЁӢи§ҒдёӢеӣҫ

еҒҮи®ҫзҺ°еңЁжңүдёӨдёӘзәҝзЁӢеҗҢж—¶жү§иЎҢдәҶStringBuilderзҡ„append()ж–№жі•пјҢдёӨдёӘзәҝзЁӢйғҪжү§иЎҢе®ҢдәҶз¬¬дә”иЎҢзҡ„ensureCapacityInternal()ж–№жі•пјҢжӯӨеҲ»count=5гҖӮ

иҝҷдёӘж—¶еҖҷзәҝзЁӢ1зҡ„cpuж—¶й—ҙзүҮз”Ёе®ҢдәҶпјҢзәҝзЁӢ2з»§з»ӯжү§иЎҢгҖӮзәҝзЁӢ2жү§иЎҢе®Ңж•ҙдёӘappend()ж–№жі•еҗҺcountеҸҳжҲҗ6дәҶ

зәҝзЁӢ1з»§з»ӯжү§иЎҢз¬¬е…ӯиЎҢзҡ„str.getChars()ж–№жі•зҡ„ж—¶еҖҷжӢҝеҲ°зҡ„countеҖје°ұжҳҜ6дәҶпјҢжү§иЎҢcharж•°з»„жӢ·иҙқзҡ„ж—¶еҖҷе°ұдјҡжҠӣеҮәArrayIndexOutOfBoundsExceptionејӮеёёгҖӮ

иҮіжӯӨпјҢStringBuilderдёәд»Җд№ҲдёҚе®үе…Ёе·Із»ҸеҲҶжһҗе®ҢдәҶгҖӮеҰӮжһңжҲ‘д»¬е°ҶжөӢиҜ•д»Јз Ғзҡ„StringBuilderеҜ№иұЎжҚўжҲҗStringBufferеҜ№иұЎдјҡиҫ“еҮәд»Җд№Ҳе‘ўпјҹ

еҪ“з„¶жҳҜиҫ“еҮә10000е•ҰпјҒ

йӮЈд№ҲStringBufferз”Ёд»Җд№ҲжүӢж®өдҝқиҜҒзәҝзЁӢе®үе…Ёзҡ„пјҹиҝҷдёӘй—®йўҳдҪ зӮ№иҝӣStringBufferзҡ„append()ж–№жі•йҮҢйқўе°ұзҹҘйҒ“дәҶгҖӮ
