RedisдҝқеӯҳдёҖдёӘз”ЁжҲ·дҝЎжҒҜжҳҜз”ЁStringиҝҳжҳҜHashпјҹ

еҸ‘иЎЁдәҺ 2020-08-14 жӣҙж–°дәҺ 2025-06-25 еҲҶзұ»дәҺ Redis

ж–№ејҸдёҖ

еҰӮжһңз”Ёжҷ®йҖҡзҡ„key/valueз»“жһ„жқҘеӯҳеӮЁпјҢдё»иҰҒжңүд»ҘдёӢ2з§ҚеӯҳеӮЁж–№ејҸпјҡ

з¬¬дёҖз§Қж–№ејҸе°Ҷз”ЁжҲ·IDдҪңдёәжҹҘжүҫkey,жҠҠе…¶д»–дҝЎжҒҜе°ҒиЈ…жҲҗдёҖдёӘеҜ№иұЎд»ҘеәҸеҲ—еҢ–зҡ„ж–№ејҸеӯҳеӮЁпјҢ

еҰӮпјҡ

иҝҷз§Қж–№ејҸзҡ„зјәзӮ№жҳҜпјҢеўһеҠ дәҶеәҸеҲ—еҢ–/еҸҚеәҸеҲ—еҢ–зҡ„ејҖй”ҖпјҢе№¶дё”еңЁйңҖиҰҒдҝ®ж”№е…¶дёӯдёҖйЎ№дҝЎжҒҜж—¶пјҢйңҖиҰҒжҠҠж•ҙдёӘеҜ№иұЎеҸ–еӣһпјҢе№¶дё”дҝ®ж”№ж“ҚдҪңйңҖиҰҒеҜ№е№¶еҸ‘иҝӣиЎҢдҝқжҠӨпјҢеј•е…ҘCASзӯүеӨҚжқӮй—®йўҳгҖӮ

ж–№ејҸдәҢ

з¬¬дәҢз§Қж–№жі•жҳҜиҝҷдёӘз”ЁжҲ·дҝЎжҒҜеҜ№иұЎжңүеӨҡе°‘жҲҗе‘ҳе°ұеӯҳжҲҗеӨҡе°‘дёӘkey-valueеҜ№е„ҝпјҢз”Ёз”ЁжҲ·ID+еҜ№еә”еұһжҖ§зҡ„еҗҚз§°дҪңдёәе”ҜдёҖж ҮиҜҶжқҘеҸ–еҫ—еҜ№еә”еұһжҖ§зҡ„еҖјпјҢ

еҰӮпјҡ

иҷҪз„¶зңҒеҺ»дәҶеәҸеҲ—еҢ–ејҖй”Җе’Ңе№¶еҸ‘й—®йўҳпјҢдҪҶжҳҜз”ЁжҲ·IDдёәйҮҚеӨҚеӯҳеӮЁпјҢеҰӮжһңеӯҳеңЁеӨ§йҮҸиҝҷж ·зҡ„ж•°жҚ®пјҢеҶ…еӯҳжөӘиҙ№иҝҳжҳҜйқһеёёеҸҜи§Ӯзҡ„гҖӮ

жңҖеҘҪзҡ„еҠһжі•

йӮЈд№ҲRedisжҸҗдҫӣзҡ„HashеҫҲеҘҪзҡ„и§ЈеҶідәҶиҝҷдёӘй—®йўҳпјҢRedisзҡ„Hashе®һйҷ…жҳҜеҶ…йғЁеӯҳеӮЁзҡ„ValueдёәдёҖдёӘHashMapпјҢ

е№¶жҸҗдҫӣдәҶзӣҙжҺҘеӯҳеҸ–иҝҷдёӘMapжҲҗе‘ҳзҡ„жҺҘеҸЈпјҢ

еҰӮпјҡ

д№ҹе°ұжҳҜиҜҙпјҢKeyд»Қз„¶жҳҜз”ЁжҲ·ID,valueжҳҜдёҖдёӘMapпјҢиҝҷдёӘMapзҡ„keyжҳҜжҲҗе‘ҳзҡ„еұһжҖ§еҗҚпјҢvalueжҳҜеұһжҖ§еҖјпјҢ иҝҷж ·еҜ№ж•°жҚ®зҡ„дҝ®ж”№е’ҢеӯҳеҸ–йғҪеҸҜд»ҘзӣҙжҺҘйҖҡиҝҮе…¶еҶ…йғЁMapзҡ„Key(RedisйҮҢз§°еҶ…йғЁMapзҡ„keyдёәfield), д№ҹе°ұжҳҜйҖҡиҝҮ

key(з”ЁжҲ·ID) + field(еұһжҖ§ж Үзӯҫ) ж“ҚдҪңеҜ№еә”еұһжҖ§ж•°жҚ®дәҶпјҢж—ўдёҚйңҖиҰҒйҮҚеӨҚеӯҳеӮЁж•°жҚ®пјҢд№ҹдёҚдјҡеёҰжқҘеәҸеҲ—еҢ–е’Ңе№¶еҸ‘дҝ®ж”№жҺ§еҲ¶зҡ„й—®йўҳгҖӮеҫҲеҘҪзҡ„и§ЈеҶідәҶй—®йўҳгҖӮ
